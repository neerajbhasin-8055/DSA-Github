DSA Practice — Java Solutions

This repository contains concise Java solutions for common data structures and algorithm problems used for practice and interview preparation. Each source file focuses on one problem and includes a short description and complexity notes.

Getting started

- Compile all solutions:

```bash
javac *.java
```

- Run a solution by specifying its class name. Example:

```bash
java LongestSubstringWithoutRepeatingCharacters
```

Included files (high level)

- 930BinarySubarraysWithSum.java — Count binary subarrays with given sum (sliding window).
- Celebrity.java — Find a celebrity using candidate elimination and validation.
- fruitBaskets.java — Longest subarray with at most two distinct items (sliding window).
- GroupAnagram.java — Group anagrams using a frequency signature + HashMap.
- Leetcode1358.java — Substrings containing 'a','b','c' (sliding window).
- LongestRepeatingCharaterInASubstring.java — Character replacement longest substring (sliding window).
- LongestSubstringWithDistinctCharacters.java — At most k distinct characters (sliding window).
- LongestSubstringWithoutRepeatingCharacters.java — Longest substring without repeating chars.
- maxConsecutiveOnes.java — Longest run of 1s after flipping up to k zeros.
- NiceSubarrays.java — Count subarrays with exactly k odd numbers (prefix counts).
- StackUsingArrays.java — Simple array-backed stack demonstration.
- StockSpan.java — Stock span using a stack of (price, span) pairs.
- subarrayWithKDifferentIntegers.java — Count subarrays with exactly k distinct integers.
- ProcessStringWithSpecialOperations.java — String processor with special commands (#, %, *).
- scripts/ — Helper PowerShell scripts: `generate_readme.ps1`, `watch_and_push.ps1`.
